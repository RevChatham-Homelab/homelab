# Purpose

This directory contains the approved Milestone Charters for the RevChatham Homelab.

A Milestone Charter defines the scope, objectives, deliverables, and completion criteria for a specific project milestone.

Each charter serves as the engineering baseline for the work performed during that milestone.

---

# Scope

Each Milestone Charter includes:

- Milestone purpose
- Objectives
- Deliverables
- Success criteria
- References
- Milestone completion criteria

---

# Lifecycle

Milestone Charters are created:

- Before implementation begins.
- After engineering planning has been completed.
- Following approval of improvements from the Project Improvement Register.

Completed charters remain in this directory as part of the project's engineering history.
