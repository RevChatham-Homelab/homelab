# Purpose

This directory contains engineering registers used to track long-term project information that evolves throughout the lifecycle of the RevChatham Homelab.

Engineering registers provide traceability between engineering decisions, milestone planning, implementation, and completed work.

---

# Current Registers

| Register | Purpose |
|----------|---------|
| `PROJECT_IMPROVEMENT_REGISTER.md` | Tracks approved engineering improvements and their implementation status. |

Additional engineering registers may be added as the project evolves.

---

# Register Management

Engineering registers are reviewed:

- At the beginning of each milestone.
- During repository engineering reviews.
- Before major version releases.
- When significant engineering decisions are made.

Registers remain living documents and are updated throughout the life of the project.
