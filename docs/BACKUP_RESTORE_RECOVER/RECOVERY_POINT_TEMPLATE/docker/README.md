# Docker Recovery Artifacts

This directory contains Docker-related recovery artifacts included in a Recovery Point.

Expected contents may include:

- Docker Compose files
- Environment file examples
- Service configuration files
- Docker volume backup archives
- Database dumps
- Container inventory
- Image and version information

The exact contents depend on the Recovery Point type and the services included.

Generated backup files should be documented in `RP_MANIFEST.md` and validated according to `VERIFICATION_STANDARD.md`.
