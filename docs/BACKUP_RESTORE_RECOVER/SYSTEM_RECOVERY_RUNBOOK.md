## Purpose

This runbook is the operational worksheet used during a system recovery.

It provides a structured checklist for documenting the recovery process, recording observations, and verifying that each recovery step has been completed successfully.

This document is intended to accompany the System Recovery Manual and the selected Recovery Point throughout the recovery process.
